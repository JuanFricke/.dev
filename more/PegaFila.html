Projeto de Sistema de Fila para o Tabelionato de Protestos

Este foi um projeto desenvolvido em conjunto com Gustavo Campos.

Objetivo e Problemas Resolvidos

O projeto teve como objetivo principal conquistar as premiações “Cartório Top” e “PQTA” para o tabelionato, além de otimizar o atendimento ao público e melhorar a eficiência do serviço.

O sistema foi criado para resolver desafios como a necessidade de melhorar a gestão das filas e reduzir o tempo de espera dos clientes. Problemas técnicos como o uso de impressoras no backend e a operação segura do servidor localmente também foram abordados, aproveitando os PCs existentes que operavam com Windows.

Critérios de Categorias de Fila

As categorias de fila foram definidas com base na legislação brasileira, que estipula a prioridade para pessoas com mais de 80 anos, garantindo conformidade com as normas e uma experiência justa para todos os clientes.

Figura 2: Interface de escolha da fila no tablet

Acessibilidade e Design

Para garantir a acessibilidade e eficiência para todos os usuários, especialmente os idosos, o design da interface do usuário foi simplificado. O botão para a fila preferencial é destacado com uma cor mais forte, facilitando a identificação por pessoas com baixa visão.

Estrutura e Benefícios

Os principais benefícios incluem a redução do tempo de espera dos clientes para menos de 15 minutos em 93% dos casos, a garantia de atendimento adequado aos idosos e a orientação correta dos clientes para o atendimento.

Figura 3: Organização das filas no sistema

Implementação e Tecnologia

O sistema utiliza PHP e JavaScript puro, com o framework BulmaCSS para o design. O banco de dados MySQL foi implementado, e o sistema roda em um servidor Nginx instalado em uma das máquinas locais do tabelionato.

A conexão com a impressora é feita via rede, e um tablet conectado via Wi-Fi permite que os usuários escolham a fila desejada. O sistema foi projetado para ter um baixíssimo delay entre o toque no botão e a retirada da senha.

Feedback e Resultados

O sistema recebeu feedback positivo de usuários e funcionários. Conquistamos as premiações “Cartório Top” e “PQTA”. As filas diminuíram, e a ocorrência de clientes pulando a frente foi reduzida.

Com a dashboard integrada, conseguimos analisar a performance dos funcionários, identificar os mais eficientes e dedicar-se ao atendimento, e acompanhar o número de atendimentos em diferentes períodos.

Figura 4: Dashboard com análise de desempenho dos funcionários

Medição do Sucesso

  • Conquista dos prêmios “Cartório Top” e “PQTA”.
  • 93% dos atendimentos realizados em menos de 15 minutos.
  • 74% dos atendimentos realizados em menos de 8 minutos.
  • Melhora na gestão de atendimento dos funcionários com análise detalhada de performance.
